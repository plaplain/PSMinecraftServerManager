# PS Minecraft Server Manager
This PowerShell module allows you to install, backup, update, and run a
Minecraft PaperMC server.

This is a personal project, I thought I'd setup a public repo on GitHub to help
me with my AZ-400 exam, and also bonus if anyone finds it useful.

## How this came about?
### The problem
My nephews play Minecraft a lot, however as they are kids, they don't have
money to rent a server. This meant that the person hosting would have to be
online, and often we weren't all free at the same time, meaning people would
get left behind.

### The idea
I had a thought, and then decided why create a little project which allows them
to run the server on their computer. I wanted to make it straight forwards for
them, and less work for me!

#### How we use it
We now use this module on our own computers to host the server, and utilise
cloud storage to sync the files between our computers. Only one person runs the
server at a time, and we just use a discord bot to see if anyone has the server
running or not.
